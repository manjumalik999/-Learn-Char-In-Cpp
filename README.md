# -Learn-Char-In-Cpp
🚀 C++ Character Variables Demo

This project is a simple C++ program that introduces the char data type. It demonstrates how to store and display characters such as letters and symbols using cout.

📚 Concepts Covered
🔤 char data type
📝 Variable declaration & initialization
🖥️ Console output using cout
📏 Output formatting with endl
💡 Basic C++ program structure

✨ Features
✅ Stores a student's grade (A)
✅ Stores a class section (C)
✅ Stores a special character (@)
✅ Displays all values in a clean and readable format

🖥️ Example Output
Grade = A
Section = C
Character = @

👨‍💻 C++ Beginners
